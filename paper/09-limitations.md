# 9. Limitations

This repository does not claim:

- exact global optimality;
- complete worldwide source coverage;
- legal authority to replace national postal systems;
- audited production routing;
- that latitude/longitude alone is sufficient for delivery;
- that generated postal-equivalent zones remove the need for local review.

The safe claim is narrower: this repository defines compatible mathematical and
software interfaces for generating, testing, and improving postal-equivalent
zones.
