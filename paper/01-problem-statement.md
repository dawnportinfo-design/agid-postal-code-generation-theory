# 1. Problem Statement

Postal codes fail in different ways:

- some countries have no postal code system;
- some systems cover cities but not informal, island, mountain, or rural areas;
- some codes are too coarse for EC delivery;
- some codes change when administrative boundaries change;
- some regions need disaster-time temporary re-partitioning;
- some users need proof of postal-equivalent membership without exposing private
  address content.

The theory treats postal-code generation as a constrained partitioning problem
over addressable referents, delivery topology, administrative hierarchy,
population, demand, terrain, and future change.
