# 7. Optimization Objectives

Postal-code generation is multi-objective.

The main objective families are:

- minimize delivery time;
- minimize delivery cost;
- balance carrier workload;
- balance parcel volume;
- minimize ambiguity;
- maximize lifecycle stability;
- minimize code length;
- maximize error tolerance;
- minimize search complexity;
- preserve administrative compatibility;
- preserve international address compatibility.

No single objective dominates globally. The repository therefore treats a
postal-code model as a policy-selected optimizer, not a universal winner.
